# Le Châteaubriand
Team: Fairoz Aly, Cyril Efren Fabro, Jia-Yu Joy Ho, Daniel Perrotta

<p>Le Châteaubriand is a reception hall and event venue that has historically managed all bookings in person, requiring multiple meetings to collect details like dates, guest counts, menu preferences, etc. As the business grows, this process has become increasingly difficult to manage.</p>

<p> This project is a web-based booking and client management system built for the venue owners. It gives them a centralized admin dashboard to track clients and events, replacing the manual back-and-forth with a structured digital workflow. Clients can also submit event requests online through a booking form, reducing the need for initial in-person visits. </p>
